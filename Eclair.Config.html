Class: Config

Eclair. Config

new Config()

Config Core Class

Dictionnary-like class to store settings, change them, restore the defaults etc.

This is the API of the Eclair.settings object

Methods

defaults(set)

Retrieve a default value
Parameters:
Name Type Description
set String Key of the default value to retrieve

define(set, value)

Used to set some settings, a key value object can be used, or two strings as arguments if we need to define only one setting

Parameters:
Name Type Argument Description
set Object | String Object or key
value * <optional>
 value to set when the method is used inline
Example
 var conf = new Jin.Config;
 conf.define({
    key: 'value'
 );

 // Or inline style
 conf.define('key', 'value');

freeze()

Meant to be used to froze the configuration, it means that previously stored values are now default values and can be retrieved via the defaults method
