Class: LocalStorage.Slot

Eclair. LocalStorage.Slot

new LocalStorage.Slot(name, data, expire)

Slot constructor

Slot is a useful localStorage abstraction within one class

Operate beyond a LocalStoreObject instance

Parameters:
Name Type Argument Default Description
name String Name of the slot, this is also its identifier
data Object <optional>
 {} A data object to be stored
expire Number <optional>
 settings.cacheExpire in seconds
See:
  • Eclair.LocalStorage
Example
 var Slot = Eclair.LocalStorage.Slot // create a local reference to Slot

 var mySlot = new Slot("mySlot")
 // Then attach some data to this slot
 mySlot.data.record = "Hello"
 // And store it
 mySlot.store()

 // You can also directly create the slot and store the data :
 var mySlot = new Slot("mySlot", {record: "Hello"});
