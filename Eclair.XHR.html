Class: XHR

Eclair. XHR

new XHR()

XHR Component

This compenent is a simple wrapper around XMLHttpRequest, it provides utility methods to perform synchronous or asynchronous requests.

Members

async :Boolean

When the set to false, the request will be performed synchronously. (Careful, this lock the whole page on Safari).

data :Object

Query parameters

headers :Object

Additional headers to send

headersOnly :Boolean

When the set to true, the request is aborted when the headers has been received.

method :String

HTTP method to use

response :Object|String|XMLDocument|HTMLDocument

Response (defaults to null) The response entity body according to responseType, as an ArrayBuffer, Blob, Document, JavaScript object (for "json"), or string. This is null if the request is not complete or was not successful.

responseText :String

Response text (defaults to null)

responseType :String

Can be set to change the response type.

Should not be used with synchronous requests. But support is provided on synchronous requests for "document" responseType

Ex: "document", "json", "blob", "arraybuffer"

timeout :Boolean|Number

Set a timeout for request

url :String

URL of the ressource to request
