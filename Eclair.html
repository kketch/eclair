<static> ajax(options)

jQuery-like ajax method
Parameters:
Name Type Description
options Object Request options
Properties
Name Type Argument Default Description
type String <optional>
 GET HTTP method to be used. Ex: "GET"
url String request url
async Boolean <optional>
 true Should the be synchronous or asynchronous ?
success Function <optional>
 false A success callback, it receives the response text as its first argument, the second is the XMLHttpRequest instance used
error Function <optional>
 false A error callback, it receives the status text as its first argument, then the XMLHttpRequest
proxy String <optional>
 A intermediate "proxy" to be used, this string is simply prepended to the request url.
timestamp Number <optional>
 false Should a timestamp be appended to the request url to prevent caching ?
Example
 Eclair.ajax({
 	type: 'GET',
 	url: 'http://foo.com/bar',
 	success: function (response) {
 		console.log('The response is ', response)
 	},
 	error: function (status) {
 		console.log('Error: ', status)
 	}
 });
